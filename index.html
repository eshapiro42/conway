Rules
This game is based on Conway's Game of Life, the infamous cellular automaton devised by John Conway in 1970.

The rules are simple:

  • At any time, each cell on the board is either
    alive
    or
    dead
    .

  • You, the player, choose a cell. If that cell is currently alive, it will die. If it is currently dead, it will come alive.

  • The entire board is then recalculated according to the traditional rules of Conway's Game of Life.

  • You win the game when you kill all the cells. Try to do it in the fewest moves possible!

Fun fact: Since the board is generated for you randomly, there's actually a chance you could win in zero moves! But the probability of that happening is 1 in 7598742300000000, so good luck.

In case you aren't familiar with Conway's Game of Life, when the board recalculates it undergoes the following changes:

  • Any live cell with fewer than two live neighbors dies, as if caused by underpopulation.

  • Any live cell with two or three live neighbors lives on to the next generation.

  • Any live cell with more than three live neighbors dies, as if by overpopulation.

  • Any dead cell with exactly three neighbors becomes a live cell, as if by reproduction.
